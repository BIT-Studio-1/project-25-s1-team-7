# Team 7 - The Cursed Castle

A text-based escape room game built in C# for our BIT project.

## How to Play
- Type commands into the console to interact with the game
- Use `help` to see available commands
- Navigate between rooms using `move`
- Pick up items and use them to escape each room
- Each room is harder than the last!

## Commands
- `look` - display current room
- `move` - move in a direction
- `inventory` - show your inventory
- `help` - show available commands
- `quit` - exit the game
